## NGSI Go tutorial for Keyrock

## Identity Management

### Get settings of alias

```console
ngsi server get --host keyrock
```

```json
ngsi server get --host keyrock
serverType keyrock
serverHost http://keyrock:3000
IdmType idm
IdmHost http://keyrock:3000/v1/auth/tokens
Username keyrock@letsfiware.jp
Password 1234
```
